3b. Fitur Impor & Ekspor Excel Cepat

Cara Impor Sangat Mudah

Anda dapat menyalin data langsung dari Microsoft Excel atau Google Sheets, lalu menempelkannya ke kotak teks di sebelah kanan.

Format Kolom Excel untuk Siswa:
  1. NIS (Kolom 1)
  2. NISN (Kolom 2)
  3. Nama Lengkap (Kolom 3)
  4. Tempat Lahir (Kolom 4)
  5. Tanggal Lahir (Kolom 5 - Format: YYYY-MM-DD atau DD-MM-YYYY)
  6. Jenis Kelamin (Laki-laki / Perempuan)
  7. Agama
  8. Alamat
Format Kolom Excel untuk Nilai Mapel:

Cukup salin deretan nilai angka (satu atau banyak kolom). Urutan nilai akan dicocokkan otomatis berdasarkan urutan daftar siswa aktif di kelas Anda.

*Pemisah tab/kolom terdeteksi otomatis saat menyalin dari Excel.

Gunakan opsi ini untuk membuat file cadangan sistem (.json) atau memindahkan semua isian rapor dari satu laptop/komputer ke perangkat lainnya.

Peringatan: Mengimpor file database akan menimpa seluruh entri data saat ini!
